# Fake News Detector

This is a Python project for detecting fake news using ML/NLP techniques.

## Contents

- `src/` – Source code
- `data_sample/` – Small sample dataset (full dataset hosted externally)
- `models/` – Model scripts (trained models hosted externally)
- `.gitignore` – Ignored large files

## Usage

1. Clone the repo
2. Install dependencies from `requirements.txt`
3. Run scripts in `src/`

Large datasets and trained models are **not included**.
